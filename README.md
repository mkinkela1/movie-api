# movie-api

To run this app, please follow this instructions:

1. Clone repo
2. cd movie-api/
3. npm install
4. cp .env.dist .env
5. Create api key on https://www.themoviedb.org/
6. Fill missing information in .env
7. npm run dev


This project is deployed on https://the-movie-db-react.netlify.app/home
